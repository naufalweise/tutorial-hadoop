Agar bisa copy paste, gunakan ssh untuk connect dari terminal windows/mac ke vm.
1. Buka command prompt di Windows/MAC anda
2. Jalankan command berikut
```
ssh vboxuser@192.168.56.50
```
3. Masukkan password VM
